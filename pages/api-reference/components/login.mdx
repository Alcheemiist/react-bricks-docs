# Login

This component **renders the Login page** for React Bricks [Admin Dashboard](/docs/getting-started/admin-dashboard).

It needs no props, but it must be wrapped by the [Admin](admin) component to receive the needed context.

The route where this component is shown has to be specified in [ReactBricks](react-bricks)'s [configuration](/docs/getting-started/configuration) parameter `loginPath`.
