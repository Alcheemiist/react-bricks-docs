# Create your own visual editing component

Using the [`useVisualEdit`](/api-reference/hooks/use-visual-edit) hook, you can create your own visual editing component, accessing React Bricks internals to get and set a prop.

For example, you may create a code editor or have fun with canvas.

[See useVisualEdit documentation](/api-reference/hooks/use-visual-edit)
